# 🌐 Charles Proxy

Настроила **Charles Proxy** для **desktop-браузера** и **мобильного устройства**, выполнила практические сценарии по перехвату, подмене и перенаправлению трафика для <a href="https://intern.demoshopping.ru" target="_blank">intern.demoshopping.ru</a>.

В ходе работы использовала **Breakpoint**, **Rewrite**, **Map Remote** и **Map Local**:
- изменяла параметры запросов и ответов;
- моделировала ответ сервера со статус-кодом <strong>403</strong>;
- перенаправляла запросы с <strong>Prod</strong> на <strong>Stage</strong>;
- изменяла запрос на удаление товара в мобильной корзине;
- подменяла содержимое страницы в мобильном браузере;
- перехватывала HTTPS-запросы с мобильного устройства и анализировала <strong>user-agent</strong>.
